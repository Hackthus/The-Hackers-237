## Présentation

**The H@ckers 237** est un projet **open source** dédié au **Pentesting et Hacking Ethique**.

Ce projet documente des techniques utilisées dans :

- les **tests d'intrusion modernes**
- les **attaques Active Directory**
- les **attaque  Applications et Services Web**
- l'**exploitation de vulnérabilités**


## Objectifs du projet 

- centraliser des connaissances en cybersécurité offensive
- documenter des techniques réelles de pentesting
- créer une base de connaissances pour l'apprentissage
- Apporter des remédiations pour corriger et protéger les systèmes d'informations

## 🤝 Contribution

Les contributions sont les bienvenues.

Pour contribuer au projet :

1. Forker le dépôt
2. Créer une branche (`git checkout -b add/nouvelle-technique`)
3. Ajouter votre documentation en suivant le template
4. Soumettre une Pull Request

Des templates sont disponibles pour créer de nouvelles pages de manière cohérente.
