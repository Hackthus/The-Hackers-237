# Contact

Une question, une suggestion, une erreur à signaler ou envie de contribuer au projet ? Plusieurs moyens de nous contacter.  
Pour un contact direct vous pouvez écrire au mail suivant : hackers237@gmail.com

## Canaux de Contact

### GitHub Canal principal

GitHub est le canal privilégié pour toute communication liée au projet.

| Type | Action | Lien |
|---|---|---|
| Bug / Erreur | Ouvrir une Issue | [Nouvelle Issue](https://github.com/Hackthus/The-Hackers-237/issues/new) |
| Suggestion | Ouvrir une Discussion | [Discussions](https://github.com/Hackthus/The-Hackers-237/discussions) |
| Contribution | Ouvrir une Pull Request | [Guide de contribution](/contributing) |
| Question générale | Ouvrir une Discussion | [Discussions](https://github.com/Hackthus/The-Hackers-237/discussions) |


## Signaler un Problème

### Erreur dans la documentation

Si vous trouvez une erreur dans une commande, une information incorrecte ou un lien cassé :

1. Cliquer sur **Edit this page on GitHub** en bas de la page concernée
2. Corriger directement et soumettre une Pull Request
3. Ou ouvrir une [Issue](https://github.com/Hackthus/The-Hackers-237/issues/new) en décrivant le problème

### Template Issue
```markdown
 ## Description du problème
 Description claire et concise.

 ## Page concernée
 URL ou chemin de la page.

 ## Comportement attendu
 Ce qui devrait être correct.

 ## Comportement actuel
 Ce qui est incorrect.

 ## Suggestion de correction
 Si vous en avez une.
```

## Contribuer au Projet

Le projet est ouvert aux contributions nouvelles pages, corrections, améliorations du design ou de la documentation.

Consulter le [Guide de Contribution](/guide) pour les détails sur :

- Comment forker et cloner le projet
- Les templates à utiliser
- Les bonnes pratiques de rédaction
- Comment soumettre une Pull Request

Toute utilisation malveillante ou illégale des informations contenues dans ce wiki est strictement interdite et contraire au [Code de Conduite](/code-of-conduct) du projet.
